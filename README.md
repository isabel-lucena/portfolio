# portfolio
## Portfólio de Isabel Lucena
Este é um projeto de portfólio responsivo desenvolvido com tecnologias web fundamentais (HTML, CSS e JavaScript), projetado para apresentar meus projetos e habilidades de forma limpa, moderna e bonita.

## Funcionalidades

Tema claro/escuro com persistência via localStorage (feito com javascript)
Cursor personalizado com anel seguidor animado
Animações de entrada via IntersectionObserver
Formulário de contato que abre o cliente de e-mail com os dados preenchidos
Layout adaptável para mobile e desktop
Navegação suave com scroll-behavior

## Estrutura de Páginas
├── index.html        Página inicial (breve apresentação)

├── sobremim.html     Página de detalhes sobre mim e habilidades

├── projetos.html     Página de projetos e certificados

├── style.css         Estilização da página

└── script.js         Lógica de interatividade

## Tecnologias
Tecnologia e seus usos:
HTML5 - Estrutura das páginas
CSS3 - Estilização, variáveis CSS, animações
JavaScript - Interatividade, tema, cursor
Google Fonts(fontes extras utilizadas) - Cormorant Garamond e DM Sans
